Git

  • git --version - проверка наличия и версии git.
  • git config - команда конфигурации, используется для задания или изменения имени пользователя, электронной почты.
  • git init - команда для создания пустого репозитория Git или повторной инициализации существующего.
  • git status - проверка статуса репозитория, с помощью этой команды можно проверить подготовленные, неподготовленные и неотслеживаемые файлы.
  • git add - индексирование указанного файла для последующего коммита
  • git reset - отмена индексации указанного файла, при этом сохраняется его содержимое
  • git commit -m "[сообщение с описанием]" - фиксация проиндексированных изменений и сохранение их в историю версий
  • git branch [имя ветки] - создание новой ветки
  • git reset [коммит] - отмена всех коммитов после заданного, все изменения в рабочей директории остаются
